Is SnapNostr not working for you?

Checkout the following list of troubleshooting steps.
  1. Override default Nostr relays

    Sometimes, certain relays take up extra time to provide the information about Nostr notes. In such cases, you can override existing Nostr relays list in the Relays option.

  2. Try another browser

    Browsers like Opera mini, Internet explorer are not tested. So switch to tested and supported browsers like Chrome, Firefox, Edge, Safari and Brave.

  3. There is no shadow after downloading/copying image

    Chances are, you might be using Safari browser. Safari has some issues with rendering shadow, or I might not know how to handle shadow in Safari. Either it's suggested to use other supported browsers for expected results.
    If you know how to resolve this shadow problem in Safari and like to help me, I thank you in advance. Reach me out via hem@hemath.dev

  4. DP is not displayed / Image attachment in the note is not displayed

    There is a possibility that your image is under a protection wall that SnapNostr wasn't able to resolve. Sometimes Safari has issues, try different browsers. You can reach me out to debug this issue.

  5. Tried all, but no luck yet!

    If you tried all the possibilities and still face issues, I'm afraid, I might not have handled an edge case.
    Please reach me out via hem@hemath.dev. SnapNostr doesn't collect any data from you. So I can't resolve the problem without your help!
